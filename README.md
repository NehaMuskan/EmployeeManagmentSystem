# EmployeeManagmentSystem

Employee Management System includes:
1) LOGIN
2) MENU (Employee Section & Salary Section)
   # EMPLOYEE SECTION
   1. ADD EMPLOYEE
   2. UPDATE EMPLOYEE
   3. DELETE DETAILS 
   4. EMPLOYEE DETAILS
   # SALARY SECTION
   5. SALARY RECEIPT
   6. ALLOWANCE
# SOFTWARE USED:
 NetBeans 16,
 XAMPP 3.3.0
